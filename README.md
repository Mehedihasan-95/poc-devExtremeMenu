# DevExtrem WEB


## Installation 

##### Requires:

- [Node](https://nodejs.org/) version `16.10.0`
- npm version `7.24.0`
- Agular ClI : `13.3.7`

##### Install Agular CLI

````
npm install -g @angular/cli
````
##### DevExtreme CLI (if needed)
````
npm i -g devextreme-cli
````



### Run Project
- With Angular
```
ng s -o
```
- With Npm

```
npm start
```
